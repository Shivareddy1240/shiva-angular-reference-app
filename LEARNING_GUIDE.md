# Learning Guide

1. Start with `app.routes.ts` to understand navigation.
2. Open `PortfolioDataService` to understand mock data and signals.
3. Study `ProjectsComponent` for filtering, forms and computed state.
4. Study `ContactComponent` for reactive forms.
5. Study `AngularLabComponent` for binding, signals and control flow.
6. Study shared components for reusable UI patterns.
