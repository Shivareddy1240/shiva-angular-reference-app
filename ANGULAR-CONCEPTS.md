# Angular Concepts Demonstrated

## Standalone Components
All components are standalone. No NgModule is required.

## Routing
The app uses `provideRouter` and lazy `loadComponent` routes.

## Signals
Project and skill state use Angular signals. Dashboard values use computed signals.

## Forms
Contact page uses reactive forms. Learning Lab demonstrates two-way binding with `ngModel`.

## Pipes
`skillLevel` and `statusBadge` show reusable formatting.

## Directives
`appHighlight` demonstrates host listeners and DOM styling through Renderer2.

## Lifecycle Hooks
About page logs `ngOnInit`, `ngOnChanges`, and `ngOnDestroy` behavior.
