# Shiva Angular Reference Portfolio

A modern Angular portfolio and reference application demonstrating practical Angular features using a personal portfolio use case.

## Highlights

- Standalone Angular architecture
- Separate HTML/SCSS/TS files
- Routing, lazy-loaded pages, guard and resolver
- Signals, computed state and effects
- Reactive forms, template forms and validation
- Custom pipes and directives
- Toasts, modal, command palette, skeleton loaders and dark mode
- localStorage favorites, notes and contact messages
- GitHub Actions workflow, unit tests and Playwright starter tests

## Run

```bash
npm install
npm start
```

Open `http://localhost:4200`.

## Build

```bash
npm run build
```

## Test

```bash
npm test
npm run e2e
```
